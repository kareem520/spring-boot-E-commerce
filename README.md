Spring-Ecomm 🛒
A sophisticated, multi-seller E-commerce REST API built with Spring Boot 3. This project implements strict business logic for data integrity, role-based access control (RBAC), and is fully deployed on Railway.

🌍 Live Demo
Base URL: https://spring-boot-e-commerce-production-1434.up.railway.app
Interactive API Docs (Swagger): https://spring-boot-e-commerce-production-1434.up.railway.app/swagger-ui/index.html.

🛠️ Tech Stack
Framework: Spring Boot
Database: MySQL (Hosted on Railway)
Security: Spring Security & JWT Authentication
Documentation: OpenAPI 3.0 / Swagger UI 
ORM: Spring Data JPA
